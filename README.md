## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Running Webpack
First terminal window:
```sh
npm run build:dev
```

### Building/Running Docker Container
Second terminal window:
```sh
npm run docker:build
# Or below if docker image already built
npm run docker:run
```
